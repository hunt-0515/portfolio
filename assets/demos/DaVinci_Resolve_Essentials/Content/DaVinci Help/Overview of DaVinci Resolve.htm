Overview of DaVinci Resolve

DaVinci Resolve is a video editing application used to import media, edit timelines, and export finished videos in a single workflow. It combines tools for cutting, editing, audio mixing, and color correction.

Download DaVinci Resolve from the Blackmagic Design website:
https://www.blackmagicdesign.com/ca/products/davinciresolve

Use DaVinci Resolve when you want to organize media files, assemble clips into a timeline, and produce a final video for sharing or publication.

The interface is divided into workspace pages, such as Cut, Edit, and Deliver, each designed for a specific stage of the editing process.

Quick start procedure

To quickly put together a simple video, follow these steps:

  1. Create a new project
  2. Import your media files
  3. Use the source viewer to prepare your clips
  4. Create a timeline from your clips
  5. Use the Cut page to make adjustments
  6. Add a title (optional)
  7. Export your video with the Deliver Page
