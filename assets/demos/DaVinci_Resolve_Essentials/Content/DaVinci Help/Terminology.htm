Terminology

Project: a container for your media, timelines, and editing work. See: Create and Manage Projects.

Playhead: a marker that shows the current position in a timeline or clip.

Media: video, audio, or image files used in a project. See: Import Media.

Clip: a piece of a video or audio file used in a timeline. A clip can be a full file or a selected portion of it.

Source Viewer: a panel used to preview media contents and set in and out points. See: Use the Source Viewer.

Timeline: a sequence of clips arranged in the editor. See: Create a Timeline.

Timeline viewer: a panel used to preview the current timeline. See: Use the Cut Page and Use the Edit Page.

In point / Out point: the start or end positions of a clip.

Layers: stacked video or audio tracks in the timeline. See: Work with Audio and Video Layers.

Blade edit tool: a tool used to split clips in the timeline.

Trim edit tool: a tool used to non-destructively adjust the start and end of a clip in the timeline.

Render: the process of exporting a finished video file from a timeline. See: Use the Deliver Page.
